# 📋 Todo App

A simple React Native Todo App for managing daily tasks efficiently.

## 🚀 Features
- Add, update, and delete tasks
- Mark tasks as **in progress** or **done**
- Filter tasks based on their status
- Responsive and user-friendly UI

## 🛠️ Tech Stack
- React Native
- React Navigation
- Ionicons (for icons)
